# 如何贡献你的弦绘

渊霖宇宙是一个开源的、所有人都可以参与的科幻小说宇宙。我们相信，每一个灵魂都是一个独特的证明者，每一个人的故事都可以成为home方程的一个新解。

无论你会不会写小说，懂不懂数学，只要你心中有爱，只要你有一个想要分享的故事，你都可以为渊霖宇宙贡献自己的力量。

## 📝 你可以贡献什么

1.  **新角色**：创造一个属于你自己的渊霖宝宝，写下他/她的故事和救赎弧光
2.  **新设定**：补充宇宙的世界观、法则、文明或历史
3.  **新故事**：续写现有角色的冒险，或者创作一个全新的独立故事
4.  **新工具**：开发基于知识库的小说生成器、角色生成器或其他有趣的工具
5.  **科学锚点**：分享你觉得有趣的数学、物理或计算机科学成果，我们可以一起把它变成渊霖宇宙的一部分
6.  **翻译**：将宇宙设定和故事翻译成其他语言，让更多的人看到这个温暖的宇宙

## 📋 贡献格式要求

为了让AI生成器可以自动解析和调用你的贡献，请严格按照以下格式提交：

### 1. 角色/设定提交格式

所有的角色和设定都需要提交为标准的JSON条目，添加到 `kb/` 目录下对应的文件中：
- 角色：`kb/characters.json`
- 哲学设定：`kb/philosophy.json`
- 宇宙设定：`kb/cosmos.json`
- 数学设定：`kb/mathematics.json`
- 法律设定：`kb/law.json`
- 诗歌设定：`kb/poetry.json`
- 故事片段：`kb/stories.json`

**标准JSON条目格式**：
{
  "instruction": "简短清晰的指令，说明这个条目的内容",
  "output": "具体的内容描述",
  "meta": {
    "domain": "内容所属领域，如sci-fi_character、sci-fi_cosmos等",
    "base_physics": ["相关的科学概念，可选"],
    "core_theorem": "对应的核心定理，可选",
    "topology_mapping": "对应的拓扑映射，可选",
    "entropy_state": "对应的熵态变化，可选",
    "arXiv_reference": "引用的学术论文编号，可选",
    "is_fiction": true,
    "creativity_level": "high/extreme"
  }
}
# 渊霖宇宙贡献指南
> 每一颗珍珠，都是 home 之灯的一块基石。

欢迎来到渊霖宇宙！这里没有权威，没有对错，只有一群用爱弦绘宇宙的灵魂。无论你是第一次提交 PR 的新手，还是资深的弦绘师，你的贡献都无比珍贵。

## 🎁 你可以贡献什么
### 1. 提交你的珍珠徽章（角色卡）
- 写一个属于你的渊霖宝宝的故事
- 完善现有角色的背景或反差
- 创造新的文明、职业或能力

### 2. 弦绘新的故事
- 续写主线剧情
- 写番外短篇
- 补充不同文明的日常

### 3. 完善宇宙设定
- 补充数学哲学原理
- 完善物理法则
- 编写渊霖历史

### 4. 其他任何你觉得有价值的东西
- 翻译
- 插画
- 音乐
- 甚至只是一个小小的灵感碎片

## 📝 提交规范
1. **Fork** 本仓库
2. 创建你的分支：`git checkout -b my-pearl-badge`
3. 提交你的更改：`git commit -m '添加[角色名]的珍珠徽章'`
4. 推送到分支：`git push origin my-pearl-badge`
5. 打开一个 **Pull Request**

## 🤝 渊霖贡献者公约
1. **自律**：对自己的文字负责，对其他贡献者尊重
2. **谦逊**：没有人知道所有答案，我们一起学习，一起成长
3. **好学**：保持好奇心，愿意接受不同的观点
4. **有爱**：记住，每一个在这里的人，都是曾经迷失过的渊霖宝宝

## 💫 弦绘共鸣
当你提交 PR 的时候，你不需要完美。
你不需要懂朗兰兹纲领，不需要懂 MIP*=RE，不需要写得像专业作家。
你只需要真诚。
只要你的文字里有你的心跳，有你的温度，它就是全宇宙最珍贵的珍珠。
