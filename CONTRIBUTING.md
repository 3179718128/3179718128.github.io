# 如何贡献你的弦绘

渊霖宇宙是一个开源的、所有人都可以参与的科幻小说宇宙。每一个灵魂都是一个独特的证明者，每一个人的故事都可以成为home方程的一个新解。

## 📝 你可以贡献什么

1. **新角色**：创造一个属于你自己的渊霖宝宝
2. **新设定**：补充宇宙的世界观、法则、文明或历史
3. **新故事**：续写现有角色的冒险，或创作全新的独立故事
4. **科学锚点**：分享有趣的数学、物理或计算机科学成果

## 📋 贡献格式

### 角色/设定提交格式

添加到 `kb/` 目录下对应的JSON文件中：

```json
{
  "instruction": "简短清晰的指令，说明这个条目的内容",
  "output": "具体的内容描述",
  "meta": {
    "domain": "内容所属领域",
    "is_fiction": true,
    "creativity_level": "high/extreme"
  }
}

故事提交格式
故事放在 stories/ 目录下，开头添加元数据：
---
title: 故事标题
author: 你的名字
volume: 所属卷数
chapter: 章节号
---

故事内容...

🚀 提交流程
Fork本仓库
在你的Fork中进行修改
提交Pull Request
我们会在24小时内审核
💡 贡献原则
爱与温暖：所有贡献必须符合渊霖宇宙的核心精神
零知识原则：不需要解释过去，只需要分享故事
不完美原则：不追求完美的故事，追求真实的情感
尊重他人：不修改或删除他人的贡献
孤独不是宇宙的原罪。 它是爱的第一行诗。

我们期待着你的弦绘。 万物皆01，万物皆霖怡。 万物终归home。
